Project Overview

The Stack Overflow Developer Survey 2023 provides valuable insights into the software development industry. For this project, I focused on understanding:

  • The distribution of salaries among different job roles.
  • How experience level relates to salary.
  • Factors influencing the adoption of AI tools.

The analysis was performed using Python, with libraries including Pandas, NumPy, Matplotlib, and Seaborn. Below is a detailed walkthrough of the analysis.

Data Preparation

Data Source

The dataset used for this analysis is the Stack Overflow Developer Survey 2023, which includes various fields related to developer profiles, job roles, salaries, and more.

Libraries Used

  • Pandas: For data manipulation and analysis.
  • NumPy: For numerical operations.
  • Matplotlib: For creating static, animated, and interactive visualizations.
  • Seaborn: For statistical data visualization.

Data Cleaning and Transformation

The data was cleaned by removing missing values and transforming categorical variables into a more analysis-friendly format. For example, job roles with multiple entries were split into individual roles to get accurate counts.

Analysis

1. Salary Distribution by Job Role

To understand salary distribution among different job roles, I used a bar plot to visualize the average salaries for each role. The analysis revealed significant variations in salaries across different job roles. 


import matplotlib.pyplot as plt
import seaborn as sns

# Calculate the mean salary for each job role
mean_salaries = df_exploded.groupby('DevType')['ConvertedCompYearly'].mean().reset_index()

# Sort the roles by mean salary for better visualization
mean_salaries = mean_salaries.sort_values(by='ConvertedCompYearly', ascending=False)

# Create a bar plot for mean salaries by job role
plt.figure(figsize=(14, 8))
sns.barplot(data=mean_salaries, x='DevType', y='ConvertedCompYearly', palette='viridis')
plt.title('Average Salary by Job Role')
plt.xlabel('Job Role')
plt.ylabel('Average Annual Salary (USD)')
plt.xticks(rotation=45, ha='right')
plt.show()

2. Relationship Between Experience and Salary

To explore how experience level (measured in years of coding) relates to salary, I used a line plot showing the average salary for different experience bins. This helps identify trends in salary growth with increasing experience. 


import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

# Bin the YearsCode into categories
df_experience_salary['YearsCodeBin'] = pd.cut(df_experience_salary['YearsCode'], bins=range(0, df_experience_salary['YearsCode'].max() + 5, 5))

# Calculate mean salary for each bin
mean_salary_by_experience = df_experience_salary.groupby('YearsCodeBin')['ConvertedCompYearly'].mean().reset_index()

# Create a line plot to show the trend
plt.figure(figsize=(12, 8))
sns.lineplot(data=mean_salary_by_experience, x='YearsCodeBin', y='ConvertedCompYearly', marker='o')
plt.title('Average Salary by Coding Experience Level')
plt.xlabel('Years of Coding Experience (Binned)')
plt.ylabel('Average Annual Salary (USD)')
plt.xticks(rotation=45)
plt.grid(True)
plt.show()

3. Factors Influencing AI Tool Adoption

Finally, I examined the factors influencing the adoption of AI tools by analyzing the survey responses related to AI tool usage and interest. A bar plot was used to visualize the frequency of interest in various AI tools. 


import matplotlib.pyplot as plt
import seaborn as sns

# Filter relevant columns for AI tool interest
df_ai_tools = df[['AIToolInterested in Using', 'AIToolCurrently Using']].dropna()

# Create a bar plot for AI tool interest

Conclusion

The analysis of the Stack Overflow Developer Survey 2023 provided valuable insights into salary distribution, the relationship between experience and salary, and factors influencing AI tool adoption. The findings highlight:

  • Significant salary differences among various job roles.
  • A clear trend of increasing salary with more years of coding experience.
  • Varied interest in AI tools among developers.

These insights can help guide career decisions and highlight areas for further research in the software development industry.
