Data Science Blog Post

Project Overview

This blog post covers the analysis and findings from the Udacity Data Scientist Nanodegree Project, where we explore data from the Stack Overflow 2023 Developer Survey. The goal of this analysis is to gain insights into various aspects of the software development industry, including programming languages, job roles, and salary trends.

Installation

To run the code for this project, you need Python version 3.* and some libraries. To install these libraries, execute:

pip install -r requirements.txt

Project Motivation

The motivation behind this project is to analyze the Stack Overflow 2023 Developer Survey data to answer key questions such as:

  • What are the most used programming languages?
  • How do programming languages used at work relate to those people want to learn?
  • What are the salary trends across different job roles?

File Description

  • exploratory_analysis.ipynb: Notebook containing the data analysis and visualizations.
  • data/survey_results_public.csv: Stack Overflow's 2023 Annual Developer Survey data.
  • requirements.txt: List of Python libraries required for the project.

Results

The analysis provided insights into the following:

  • Trends in programming language usage and interest.
  • Salary distribution across different job roles.
  • Correlation between experience and salary.

The full results and detailed analysis can be found in the Jupyter Notebook file.
